# h5 Nimekäs

Tässä tehtävässä oli tarkoitus löytää omalle palvelimelle verkkotunnus/domain-nimi. Verkkotunnuksia voi ostaa DNS(Domain Name System)-palveluista esim. Namecheap. 

Itse käytin name.comin ilmaista kokeilua jonka sai github educationin kautta. Loin miikarajamaki.dev päätteisen domainin.
